# Java Programming Training

## 📌 About the Training

This repository contains the concepts, exercises, and basic projects completed during my Java Programming training. The training focused on understanding core Java concepts and improving problem-solving and programming skills through hands-on practice.

## 🚀 Topics Covered

* Classes and Objects
* Operators
* Data Structures and Algorithms (DSA) Basics
* Exception Handling
* String Buffer
* Time Complexity Analysis

## 📂 Projects and Exercises

### 1. Classes and Objects

* Creating classes and objects
* Constructors
* Methods and instance variables
* Object interaction

### 2. Operators

* Arithmetic Operators
* Relational Operators
* Logical Operators
* Assignment Operators
* Increment and Decrement Operators

### 3. Data Structures and Algorithms

* Arrays
* Searching Techniques
* Sorting Techniques
* Basic problem-solving approaches

### 4. Exception Handling

* Try-Catch blocks
* Multiple Catch statements
* Finally block
* Throw and Throws keywords
* Custom exceptions

### 5. String Buffer

* StringBuffer creation
* Append operation
* Insert operation
* Replace operation
* Delete operation
* Reverse operation

### 6. Time Complexity

* Big O Notation
* Constant Time – O(1)
* Linear Time – O(n)
* Logarithmic Time – O(log n)
* Quadratic Time – O(n²)
